Welcome to AranOS

Simple. Beautiful. Yours.
Version 1.0  ·  "Aurora"

Get around

Click the menu in the bottom-left to find every app, search, and shut down. The panel at the bottom shows open windows and the system tray.

What's included

  • Web — GNOME Web browser
  • Office — LibreOffice Writer, Calc, Impress
  • Creative — GIMP image editor
  • Media — VLC plays anything
  • Files, Calculator, Screenshots & more

Connect

Click the network icon in the system tray (bottom-right) to join Wi-Fi. Bluetooth, sound and display settings live in System Settings.

Make it yours

Open System Settings → Themes & Backgrounds to restyle everything. This is a live session — changes reset on reboot unless you install AranOS.

You're running the AranOS live session. Enjoy the tour — close this window any time.
